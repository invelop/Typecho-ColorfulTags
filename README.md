# Typecho-ColorfulTags
彩色3D标签云插件，原生JS实现。

![效果图](https://blog.irow.top/usr/uploads/2019/08/21217110.png)

## 食用方法

解压后重命名文件夹为ColorfulTags，激活插件，在插件设置中设置标签云容器ID，如果没有就手动添加一个再到后台完成设置。

具体可参考[handsome主题安装方式](blog.irow.top/archives/396.html)
